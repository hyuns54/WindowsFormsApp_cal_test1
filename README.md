
  ## 계산기 프로그램
    
  
1️⃣ **구현 기능 소개** : <br> 
       a) 기본 사칙연산 지원 <br>
			 b) 실시간 수식 구성 <br>
			 c) 결과 계산 및 출력<br>
			 d) 소수점 입력 가능<br>
			 e) 전체 초기화 및 입력 초기화 기능

2️⃣ **디자인 구조** : 

버튼 및 텍스트 박스를 이용 -> 각 버튼의 기능은 코드에서 이벤트 핸들러로 연결
계산 처리 -> 상태관리 

3️⃣ **코드 주요 부분 설명** : 숫자 및 연산자 입력 - 수식 계산 - 디스플레이 업데이트 - 소수점, 초기화 

4️⃣**담당 역할** : 
      현승 - 기본 사칙 연산 <br>
      진영 - UI 구현 / 입력 처리 및 이벤트 핸들러 구현
